# 🚀 Smart To-Do Pro

A modern, responsive To-Do Web Application built using **HTML, CSS, and JavaScript**.

## ✨ Features

* ✅ Add, edit, delete tasks
* 📅 Calendar-based date selection
* ⏰ Time scheduling
* 🔥 Priority levels (High / Medium / Low)
* ⚠️ Overdue task detection
* 📊 Circular progress tracking
* 💾 LocalStorage support
* 📱 Fully responsive design

## 🛠️ Tech Stack

* HTML5
* CSS3 (Glassmorphism + Responsive Design)
* Vanilla JavaScript
* LocalStorage API
